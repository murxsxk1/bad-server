## Подготовка к работе

1. Склонировать репозиторий
2. Запустить docker

```bash
docker compose up -d
```

3. Наполнить базу данных
[README.md](.dump%2FREADME.md)
2. Перейти по адресу <http://localhost/> и на странице должны быть продукты.
3. На странице <http://localhost/login/> можно авторизоваться.
4. Админка находится по адресу <http://localhost/admin/>
